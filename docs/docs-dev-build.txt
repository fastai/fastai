*** How to maintain http://docs-dev.fast.ai/ ***

fastai/docs/ is automatically mapped to http://docs-dev.fast.ai/ so any changes committed to git will instantly appear on http://docs-dev.fast.ai/ 

* When a file is added/removed/renamed must adjust the sidebar: docs/_data/sidebars/home_sidebar.yml

* theme/stylesheets:

- style sheet is assets/css/style.scss
- supported themes https://pages.github.com/themes/
- currently using: https://pages-themes.github.io/cayman/ - see that url for extra customizations
- guide for jekyll themes: https://idratherbewriting.com/documentation-theme-jekyll/
